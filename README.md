# Projeto DIO - Uso de IA no Desenvolvimento de Sistemas (Nisshinbo)

## Contexto e Objetivos

Este projeto tem como tema o uso de **Inteligência Artificial no desenvolvimento de sistemas**, aplicado ao contexto profissional na empresa Nisshinbo.

Durante minha atuação, participei do desenvolvimento de diferentes soluções internas, incluindo:
- Site novo e revitalizado;
- Leitor de XML com cálculo de impostos;
- Sistema de abertura de chamados;
- Sistema interno de RH.

### Objetivos de estudo

- Entender como a IA pode auxiliar no desenvolvimento de sistemas;
- Melhorar produtividade na escrita de código;
- Utilizar IA para análise, correção e otimização de sistemas;
- Criar um guia prático para uso no dia a dia.

## Engenharia de Prompts e Cicatrizes

Durante o desenvolvimento dos sistemas, utilizei IA para acelerar tarefas e melhorar a qualidade do código.

### Prompt 1
"Crie uma função em Python para ler um XML de nota fiscal e extrair os principais dados."

**Resultado:**  
A IA gerou uma base funcional, mas foi necessário ajustar para lidar com diferentes estruturas de XML.

---

### Prompt 2
"Explique como calcular impostos a partir de dados de uma nota fiscal."

**Resultado:**  
A resposta ajudou na lógica, mas não trouxe regras específicas do Brasil, exigindo adaptação manual.

---

### Prompt 3
"Melhore esse código para ficar mais organizado e performático."

**Resultado:**  
A IA ajudou na refatoração, melhorando legibilidade e estrutura.

---

### Prompt 4
"Crie um sistema simples de abertura de chamados com boas práticas."

**Resultado:**  
Gerou uma base útil, mas foi necessário adaptar para o contexto interno da empresa.

---

### Dificuldades encontradas

- Respostas genéricas sem contexto específico;
- Código que nem sempre funcionava direto;
- Necessidade de ajustes para regras de negócio;
- IA não considera totalmente o ambiente real do sistema.

---

### Aprendizados

- Prompts mais específicos geram melhores resultados;
- Informar contexto melhora muito a resposta;
- IA é excelente para acelerar, mas não substitui validação;
- Uso ideal é como apoio, não solução final.

## Miniguia de Estudo

### Resumo Estruturado

A Inteligência Artificial pode ser utilizada no desenvolvimento de sistemas como ferramenta de apoio para:

- Geração de código;
- Correção de erros;
- Explicação de lógica;
- Refatoração;
- Sugestão de melhorias.

Seu uso aumenta a produtividade, mas exige validação humana.

---

### Glossário

- **IA (Inteligência Artificial):** tecnologia que simula capacidades humanas.
- **Prompt:** instrução enviada para a IA.
- **Refatoração:** melhoria do código sem alterar funcionalidade.
- **Automação:** execução automática de tarefas.
- **Debug:** processo de identificar e corrigir erros.

---

### Prompts Reutilizáveis

"Explique esse código passo a passo"

"Identifique possíveis erros nesse código"

"Melhore esse código mantendo a lógica"

"Crie uma função para [tarefa específica]"

"Crie testes unitários para esse código"

"Explique como otimizar esse sistema"

---

## Conclusão

O uso de IA no desenvolvimento de sistemas se mostrou uma ferramenta importante para aumentar produtividade e melhorar a qualidade das soluções.

A aplicação prática nos sistemas desenvolvidos na Nisshinbo demonstrou que, com prompts bem estruturados, é possível acelerar tarefas e apoiar decisões técnicas, mantendo sempre a validação humana como etapa essencial.
